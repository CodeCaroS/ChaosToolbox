# Facts

- The app uses a Command Center direction: the home screen focuses on daily work and Second Brain Git state.
- Second Brain Git is reachable from top-level navigation instead of only being hidden inside Sources.
- The dashboard shows Second Brain Git status and gives direct access to the main Git workflow.
- The Second Brain Git screen uses a guided panel with a primary Sync action, commit message input, Pull, Commit, Push, and the existing guarded force push.
- Notes and Sources layouts are made easier to scan without changing their backend APIs or data model.
